Power tree schematic

배선도 — PCB 레이아웃

Routed board · 2-layer
TOP · F.Cu (앞면) — 24V 전원 트레이스 + 부품
BOTTOM · B.Cu (뒷면) — GND 평면 + 신호

구조: 앞면(F.Cu)=24V 폴리곤 + 전원 트레이스(메인 4mm·브랜치 2mm), 뒷면(B.Cu)=GND 솔리드 평면 + 신호선. 2oz 구리. 자동배선 후 GND 보강(도그렉·비아) 수동 추가. 벡터 원본: 02_pcb_layout.pdf / .svg.

참고 · 확인 필요

  • 극성 절대 확인: J1 배터리 + / − 를 반대로 꽂으면 보드 손상. XT60 극성키로 방지하되 조립 시 재확인.
  • 공통 접지: 모든 아두이노·드라이버의 GND는 이 보드 GND와 공통이어야 함.
  • 잠정값(현장 실측 후 확정): F1(BMS 연속전류 문의), F4(스테퍼 입력전류), 서보 스톨전류, 7S 만충전압. — 퓨즈값 미세조정용, 배치/배선과는 무관.
  • 열: 메인 4mm 트레이스는 상시 16–19A에서 약 26–37°C 상승(허용 범위). 더 시원하게 하려면 본선 양층 병렬 가능.
