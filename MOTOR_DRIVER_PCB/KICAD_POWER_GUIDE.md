# PCB_24V_CONVERT — 전원분배 보드 설계 가이드

> 신호보드(ARDUINO_MDD10A_PCB)와 같은 "직접 만들고 → Claude가 kicad-cli로 검증" 루프로 진행.
> 이 보드는 **대전류(최악 16~19A)** 를 다루므로 아래 6개 방어설계가 필수다.

---

## 이 보드의 정체
"퓨즈 분배 + 벅 인터페이스" 보드. 능동회로는 거의 없고 **터미널 + 퓨즈 + 벅모듈 + 보호부품**의 조합.
- 배터리 24V 입력 → 메인스위치/인러시/메인퓨즈 → **24V 폴리곤 버스**
- 버스에서 브랜치 퓨즈로 갈라져 스크류 터미널 출력: 모터드라이버×2, 스테퍼, 그리고 벅 3개 입력
- 벅 3개(12V 서보 / 19V 젯슨 / 5V 아두이노)는 보드 옆 장착, 터미널로 연결
- 벅 출력이 보드 출력 터미널로 돌아와 보호부품(캡/TVS) 거쳐 최종 부하로

**설계 확정(2026-07):** 젯슨 19V = 자리 예약(지금 미실장). 아두이노 = 독립 5V 벅(젯슨 리셋돼도 데드맨 유지). 모터전원은 이 보드 경유하되 배터리 직결 성격(퓨즈·스위치만) 유지 — MDD10A 회생전류 흡수 위해 사이에 벅 금지.

---

## 레일(넷) 구조
| 넷 | 전압 | 부하 |
|----|------|------|
| `V24` | 24V | 모터드라이버 2개, 스테퍼 3개, 벅 3개 입력 |
| `V12` | 12V | 허큘렉스 서보 2개 (XL4016 출력) |
| `V19` | 19V | Jetson Orin Nano (예약, 미실장) |
| `V5` | 5V | Arduino Mega (독립 급전) |
| `GND` | 0 | **스타 그라운드 — 배터리 음극 단일점** |

---

## ★ 6개 방어설계 (대전류 보드 필수)
1. **스타 그라운드** — 모든 GND를 배터리 음극 한 점에서 모음. 모터 리턴(10~20A)과 로직/UART 리턴을 그 점까지 분리. 2oz GND 폴리곤 + 비아 스티칭.
2. **2oz 구리 + 폴리곤** — 24V 버스와 모터 브랜치는 트레이스 대신 폴리곤. 10A 경로 폭 >=3.6mm.
3. **브랜치 퓨즈 7개** — 사고 격리 + BMS 단일트립 회피.
4. **인러시 리미터** — 메인 입력에 NTC 서미스터(전원 ON 돌입전류 제한, BMS 트립 방지).
5. **벌크 커패시터** — 24V 버스 1000uF/35V, 각 벅 입출력 캡. 모터 sag를 캡이 먼저 흡수.
6. **서보 벅 출력 TVS + 벌크캡** — 서보 회생전류로 12V 레일 과전압 클램프.

## 확정 부품표 (심볼 / 풋프린트)
심볼은 신호보드처럼 제네릭으로 놓고 풋프린트는 나중에 지정.

| Ref | 부품 | 심볼 | 퓨즈/정격 |
|-----|------|------|-----------|
| J_BATT | 배터리 24V 입력 | Conn_01x02 (→ M4 볼트패드 풋프린트) | — |
| SW1 | 외부 메인스위치 연결 | Conn_01x02 | — |
| RT1 | NTC 인러시 리미터 | Thermistor | — |
| F0 | 메인 퓨즈 | Fuse | 25-30A 블레이드 |
| C1 | 24V 버스 벌크 | CP (1000uF/35V) | — |
| F1 / F2 | MDD10A #1 / #2 | Fuse ×2 | 각 15A |
| J_MDD1 / J_MDD2 | 모터드라이버 24V 출력 | Conn_01x02 ×2 (24A급 터미널) | — |
| F3 | 스테퍼 | Fuse | 7.5A |
| J_STEP | 스테퍼 24V 출력 | Conn_01x02 (외부 분배) | — |
| F4 | 12V 벅 입력 | Fuse | 7.5A |
| U1 | XL4016 (24->12V) 모듈 | Conn_01x04 (IN+ IN- OUT+ OUT-) | — |
| C2 / C3 | 12V 벅 입력/출력 벌크 | CP ×2 | — |
| D1 | 서보레일 TVS | D_TVS (예: SMBJ13A) | — |
| J_12V | 서보 12V 출력 | Conn_01x02 | — |
| F5 | 19V 벅 입력 | Fuse | 5A |
| U2 | 젯슨 벅 (24->19V) 예약 | Conn_01x04 (미실장) | — |
| J_19V | 젯슨 19V 출력 | Conn_01x02 | — |
| F6 | 5V 벅 입력 | Fuse | 5A |
| U3 | 5V 벅 (24->5V) 모듈 | Conn_01x04 | — |
| J_5V | 아두이노 5V 출력 | Conn_01x02 | — |

퓨즈 총 7개: F0(메인) + F1,F2,F3(모터측) + F4,F5,F6(벅측). 아두이노(F6)는 안전상 반드시 독립.

---

## 넷 연결 규칙
```
J_BATT.1(+) → SW1 → RT1 → F0 → [V24 버스] → C1(→GND)
[V24] ─┬ F1 → J_MDD1.1
       ├ F2 → J_MDD2.1
       ├ F3 → J_STEP.1
       ├ F4 → U1.IN+ ;  U1.OUT+ → C3/D1 → J_12V.1 (=V12)
       ├ F5 → U2.IN+ ;  U2.OUT+ → J_19V.1 (=V19)
       └ F6 → U3.IN+ ;  U3.OUT+ → J_5V.1  (=V5)
GND: J_BATT.2 = 스타점. 모든 .2(GND)와 U*.IN-/OUT-, C*, D1 캐소드가 여기로.
```

---

## Phase 진행 (신호보드와 동일 루프)
- **Phase 1a** — 입력 척추: J_BATT, SW1, RT1, F0, C1 배치 + V24/GND 라벨. 저장 → 검증.
- **Phase 1b** — 모터 브랜치: F1,F2,F3 + J_MDD1,J_MDD2,J_STEP.
- **Phase 1c** — 벅 3개 브랜치: F4~F6 + U1,U2,U3 + C2,C3,D1 + J_12V,J_19V,J_5V.
- **Phase 2** — 라벨 배선(V24/V12/V19/V5/GND). Phase 3 ERC. Phase 4 풋프린트(터미널/퓨즈홀더/CP).
- **Phase 5~9** — F8 → 배치(스타그라운드!) → **2oz 넷클래스 + 폴리곤** → DRC → 2oz 거버 발주.

검증: 각 저장 후 Claude가 `kicad-cli sch export netlist`로 넷 전수 대조 + ERC.
